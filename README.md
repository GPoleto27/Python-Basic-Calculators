# Tkalculator
Basic calculator made using Tkinter (Python) for studying purposes

<h3>To run:</h3>
<ul>
  <li>
    Install Tkinter, if you haven't already
    <code>
      pip install tk
    </code>
  </li>
  <li>
    Give run permission to the code
    <code>
      sudo chmod 777 ./Tkalculator.py
    </code>
  </li>
  <li>
    Run
    <code>
      ./Tkalculator.py
    </code>
  </li>
</ul>
